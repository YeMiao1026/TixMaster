Microservices Design

  • Frontend: Next.js (Static Assets).
  • Backend: Node.js API (Business Logic).
  • Database: PostgreSQL / MongoDB Atlas (Auto-Scaling).
  • Auth: Auth0 Identity Provider (Security).

Key Decision: Separation of Concerns. UI is distinct from Logic, and Data integrity is ensured by ACID compliant databases.
