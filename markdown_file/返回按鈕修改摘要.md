## 🎨 設計風格

### 統一的設計元素
- **圖示**：所有返回按鈕都使用 `←` 箭頭
- **文字**：統一使用「返回首頁」
- **動畫**：懸停時都有平滑的過渡效果
- **響應式**：所有按鈕都適配不同螢幕尺寸

### 兩種主要樣式

#### 1. 固定位置按鈕（登入/註冊頁面）
```css
.back-button {
    position: fixed;
    top: 20px;
    left: 20px;
    background: rgba(255, 255, 255, 0.9);
    /* ... */
}
```
**使用頁面**：
- login.html
- register.html
- admin-login.html

#### 2. 導航欄整合按鈕
```css
.navbar .back-link {
    color: #6b7280;
    /* ... */
}
```
**使用頁面**：
- checkout.html
- admin-dashboard.html

#### 3. Hero 區域按鈕
```css
.back-button {
    position: absolute;
    top: 20px;
    left: 20px;
    /* ... */
}
```
**使用頁面**：
- event-detail.html

## 📱 使用者體驗改進

### 改進前
- 使用者在深層頁面（如結帳頁、活動詳情）時，需要使用瀏覽器的返回按鈕
- 沒有明確的導航路徑回到首頁
- 某些頁面（如登入頁）進入後無法輕鬆返回

### 改進後
- ✅ 每個頁面都有清晰的返回首頁路徑
- ✅ 按鈕位置明顯，易於發現
- ✅ 統一的視覺語言，提升品牌一致性
- ✅ 減少使用者的迷失感

## 🔄 導航流程

```
首頁 (index.html)
  ├─ 活動詳情 (event-detail.html) ← 可返回首頁
  │    └─ 結帳頁面 (checkout.html) ← 可返回首頁或活動頁
  ├─ 登入頁面 (login.html) ← 可返回首頁
  │    └─ 登入要求頁 (login-required.html) ← 可返回首頁
  ├─ 註冊頁面 (register.html) ← 可返回首頁
  ├─ 管理員登入 (admin-login.html) ← 可返回首頁
  └─ 管理員後台 (admin-dashboard.html) ← 可返回首頁
```
